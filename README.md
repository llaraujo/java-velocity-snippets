# Java Velocity Snippets for Sublime Text

## Vars

### [set] set var

```java
#set($${1:var}=${2:""})
```

### [var] use var

```java
$!{${2:var}}
```

## Array
### [aa] array.add

```java
$!{${1:arrayName}.add($!{${1:arrayName}.size()}, ${2:value})}
```

### [ar] array.remove

```java
remove(${1:numericKey})
```

## Conditionals

### [if] if

```java

	${2}
#end
```

### [ife] if-else

```java

#end
```

## Loop

### [fe] forEach

```java
#foreach ($${1:item} in $!{${2:myCollection}})
	${0}
#end
```

### [fe] forEach Numeric

```java
#foreach ($${1:item} in [${2:0}..${3:1}])
	${0}
#end
```

### [fi] for in

```java
#foreach ($${1:item} in $!{${2:myCollection}.keySet()})
	${0}
#end
```
